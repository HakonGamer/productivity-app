Productivity V2 — All-in-One Web App for Entrepreneurs


A powerful, offline-first productivity suite built for entrepreneurs and solopreneurs who need to manage their work, finances, and documents — all in one place, without an internet connection.



🔗 Live Demo: harmonious-kleicha-91d05f.netlify.app


What is this?

Productivity V2 is a fully client-side web application that runs entirely in the browser — no server, no account, no subscription. Everything is saved locally on your device. It was designed for entrepreneurs who want a fast, private, and reliable tool that works even without internet access.


Features

💰 Finance Tracker


Set a custom budget
Log and track expenses (up to 10 entries)
Visual summary of spending vs. budget
Real-time balance display


✅ Missions & Tasks


Add and manage tasks with deadlines
Built-in countdown timer and stopwatch
Alarm system for important reminders


📅 Calendar


Monthly calendar view
Add and manage custom events per day
Event list for selected dates


📝 Office Lite — Document Suite


Writer: Rich text editor with font size controls, save/load and PDF export
Calc: Spreadsheet editor (up to 20×10), autosum, CSV and PDF export
Slides: Presentation builder with image support, slide management and fullscreen presentation mode


🤖 AI-Powered OCR Scanner


Upload any image (JPG, PNG, WEBP)
Extract text using AI automatically
Send extracted text directly to Writer or export as PDF


🛠️ Tools & Export


Export Writer documents to PDF
Convert local files (.txt, .docx, .jpg, .png) to PDF
Export active Calc tables to PDF


📁 File Manager


View and manage all saved documents, spreadsheets and presentations
All files stored locally in the browser


⚙️ Settings


Custom accent color for full UI personalization
Language support: Spanish, English, Chinese
Full data export and restore via JSON backup
Local data reset option



Tech Stack


HTML5 — structure and layout
CSS3 — custom theming and responsive design
Vanilla JavaScript — all logic, no frameworks
LocalStorage API — offline data persistence
AI integration — OCR text extraction from images
PWA-ready — installable as a desktop/mobile app



Offline First

This app requires zero internet connection after the first load. All data is stored directly in your browser using LocalStorage. No accounts, no cloud, no tracking.
